# Downloads Directory

This directory is for placing manually downloaded dataset ZIP files.

## Usage

1. Download the dataset from:
   https://universe.roboflow.com/z-algae-bilby/beetle/dataset/1

2. Place the downloaded ZIP file in this directory

3. Run the setup script:
   ```bash
   python3 download_dataset.py
   ```

## Note

- ZIP files in this directory are ignored by git
- Only this README file is tracked
